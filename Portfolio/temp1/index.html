Wie ben ik?

Wie ben ik dat is de vraag. Nou mij naam is Blen Michil ik ben 18 jaar en zit op het Media College Amsterdam. Ik doe nu de opleiding Mediadeveloper, op deze opleiding leer je hoe je met code omgaat en hoe je websites en applicaties maakt. Waarom ik deze opleiding heb gekozen is omdat mijn oom ook een developer is, hij werkt nu bij Metro en via hem leerde ik wat je allemaal kan doen met code. En ik vond dat erg interessant vooral om dat ik toen al wist dat ik later met computers wilde gaan werken. Ik heb veel geleerd op school maar ook thuis. Ik kan goed omgaan met HTML en CSS, ik kan ook met Javascript en PHP omgaan maar dat kan nog beter. Ik gebruik soms ook Bootstrap maar dat is heel weinig ik leer nu ook om te werken met laravel. Ik leer steeds iets meer elke dag. Mijn droom voor later is dat ik een fullstackdeveloper ben, want dan kan ik veel meer alleen en dan ik ook veel meer mensen helpen. Als je meer wilt weten over mij of contact met mij wilt zoeken kan je dat doen via naar benden te scrollen of via de navigatiebar bovenin bij contact.
