Wie ben ik?

Mijn naam is Blen Michil en ik ben 18 jaar, ik ben geboren in Nederland op 28 juli 2001, mijn moeder werkt voor een vliegticket bedrijf en mijn vader heeft zijn eigen sportschool, ik ben kwart Nederland kwart Filipijns en half Eritrea’ s. ik zit nu op het Media College Amsterdam. Daar doe ik de opleiding media Developer. Waarom ik deze opleiding heb gekozen is omdat ik het interessant vond wat je allemaal met code kan, mijn oom is ook een developer en door te zien wat hij deed had ik gekozen om ook een developer te worden. Mijn sterkste punt nu is Front-End, ik kan goed om gaan met html en CSS, mijn javascript moet nog wel een beetje bij gewerkt worden. Maar dat komt nog. In backend ben ik wat minder want het eerste jaar was ik veel meer bezig met front-end dan backend. Ik weet de basics van PHP en ben net begonnen met laravel en Java. Ik hoop dat ik met jullie verder meer kan leren. Wil je meer weten of met mij in Contact zijn. Klik dan op contact in de navigatiebar. Wil je mij cv zien klik op de button.
